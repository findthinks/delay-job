## Delay-job 延迟任务调度系统
Delay-job是一个开源的延迟任务调度实现，具有分布式、轻量级特点。

### 特性
* 支持特定事件发生后，延迟一定时间后触发一次调度
* 支持分布式分片延迟任务调度
* 支持可视化管理任务分片
* 支持可视化管理延迟任务 
* 可快速部署(只依赖Mysql)

### 性能
性能测试：秒级精确调度50000次/s
测试资源：CentOS7.9，2CPU，4GB Ram运行于腾讯云虚拟机
